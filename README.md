# FinSecure – Fraud Detection API

FinSecure is a Machine Learning–powered **Fraud Detection API** built using **FastAPI**, designed to evaluate financial transactions and classify them into risk levels.

This API uses an XGBoost model trained on behavioral, transactional, device, and geolocation features.

---

##  Features

✔ Predict fraud risk for **single** transaction  
✔ Predict fraud risk for **multiple** transactions (batch mode)  
✔ Fully typed request schema using Pydantic  
✔ Clean numpy → Python type conversion  
✔ Easy integration with Postman, mobile apps, or backend systems  

---

## Tech Stack

- **FastAPI**
- **Uvicorn**
- **Pandas**
- **Joblib**
- **Scikit-Learn**
- **XGBoost**
